
IconEditor - редактор иконок

На данный момент сохраняет только в bmp-файл
На Linux не открываются ico-файлы

При изменении тон/яркость/насыщенность каждый движок сбрасывает предыдущее, то есть если изменён тон, то яркость сбросит тон. Чтобы это не происходило надо сначала закрыть окно регулировки чтобы текущий тон сохранился на холсте, а потом регулировать другие. Пока так.
