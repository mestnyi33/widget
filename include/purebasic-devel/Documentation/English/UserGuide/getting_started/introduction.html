Getting Started: Introduction

Introduction

Welcome to the PureBasic User Guide, the section of the manual aimed at guiding both novice programmers and newcomers to PureBasic through the language. Although this user guide should be readable by novice programmers we hope that more experienced programmers would still take the time to look - you should be able to make quick progress through it.

The User Guide will start from the simplest parts of the PureBasic language and work its way through the various topics and commands using examples and explanations of those examples to give you a full appreciation of writing software in PureBasic. The User Guide will explain why, when and where you need to use different pieces of code and what you are doing when you use that code.

This first chapter is an introduction to both PureBasic (in terms of a piece of software, not specifically looking at writing software just yet) and the rest of this User Guide. In the following topics, you will find explanations of how to buy, install, run and use PureBasic. The layout of the rest of the User Guide will also be described so you know what to expect.

At the bottom of every page in the User Guide you will find three links. The first takes you back to the previous topic in a chapter. The third takes you onto the next topic in a chapter. The second link returns you to the contents page. To move onto the next topic in this introduction, click on the "User Guide overview" link below.
