@Title Formules mathématiques (géométrie 2D)

@Section Introduction

  Ci-dessous vous trouverez un ensemble de diverses formules mathématiques. Données pour référence, 
  elles seront également utiles, pour combler vos éventuelles lacunes. 
  Car sans une bonne connaissance des mathématiques, 
  les jeux et les applications avancées ne peuvent pas être réalisés. . @LineBreak
  @LineBreak
  Note: A la place du symbole '^' Utilisez plutôt la commande PureBasic Pow() de la bibliothèque 'Math'. 
  'PI' réfère à la valeur constante "PI = 3.1415...". 
  Un exemple pour réaliser une telle formule en langage PureBasic : A = PI*r^2 devient A = PI*Pow(r,2)

@Section Carré

  @Image formula01.gif

  l = Longueur du côté  @LineBreak
  e = Diagonale (mesure)  @LineBreak
  A = Surface  @LineBreak
  U = Périmètre  @LineBreak
  @LineBreak
  @Bold Surface : A = l^2  @LineBreak
  @LineBreak
  @Bold Périmètre : U = 4*l  @LineBreak
  @LineBreak
  @Bold "Diagonale (mesure)" : e = SQR(2)*l  @LineBreak

@Example
@Code
  ; Longueur d'un côté du carré
  Longueur = 35 

  ; Racine de 2
  #RacineDeDeux = 1.414213538

  ; Surface
  Surface = Pow(Longueur, 2) ; ou encore Longueur * Longueur (c'est plus rapide)

  ; Périmètre
  Perimetre = 4 * Longueur 

  ; Longueur de la Diagonale 
  Diagonale = Sqr(2) * Longueur ; Pour accélérer le calcul il est possible de remplacer Sqr(2) par #RacineDeDeux 
  
  ;Affiche le résultat
  Debug "Longueur d'un côté du carré = " + Str(Longueur) + " m"
  Debug "Diagonale du carré = " + Str(Diagonale) + " m"  
  Debug "Surface du carré = " + Str(Surface) + " m²"
  Debug "Périmètre du carré = " + Str(Perimetre) + " m"
@EndCode

@Section Polygone

  @Image formula07.gif
  
  l = Longueur du côté @LineBreak
  d = Diamètre du cercle inscrit @LineBreak
  D = Diamètre du cercle circonscrit @LineBreak
  n = Nombre de côtés @LineBreak
  alpha = Angle au centre @LineBreak
  beta = Angle au sommet @LineBreak
  A = Surface @LineBreak
  U = Périmètre @LineBreak
  @LineBreak
  @Bold "Diamètre du cercle inscrit" : d = SQR(D*D-l*l) @LineBreak
  @LineBreak
  @Bold "Diamètre du cercle circonscrit" : D = SQR(d*d+l*l) @LineBreak
  @LineBreak
  @Bold "Longueur du côté" : l = D*SIN(180/n) @LineBreak
  @LineBreak
  @Bold "Angle au centre" : alpha = 360/n @LineBreak
  @LineBreak
  @Bold "Angle au sommet" : beta = 180-alpha @LineBreak
  @LineBreak
  @Bold Surface : A = n*l*d/4 @LineBreak
  @LineBreak
  @Bold Périmètre : U = l*n @LineBreak

@Section Section de cercle 

  @Image formula10.gif

  r = Rayon @LineBreak
  D = Diamètre @LineBreak
  l = Longueur de la corde @LineBreak
  lb = Longueur de l'arc @LineBreak
  alpha = Angle au centre @LineBreak
  b = Largeur @LineBreak
  A = Surface @LineBreak
  U = Périmètre @LineBreak
  @LineBreak
  @Bold "Longueur de la corde" : @LineBreak
  l = 2*r*SIN(alpha/2) @LineBreak
  l = 2*SQR(b*(2*r-b)) @LineBreak
  @LineBreak
  @Bold "Longeur de l'arc" : lb = PI*r*alpha/180 @LineBreak
  @LineBreak
  @Bold Largeur : @LineBreak
  b = l/2*TAN(alpha/4) @LineBreak
  b = r-SQR(r^2-l^2/4) @LineBreak
  @LineBreak
  @Bold Rayon : r = b/2+l^2/(8*b) @LineBreak
  @LineBreak
  @Bold Surface : @LineBreak
  A = PI*d^2*alpha/1440-(l*(r-b))/2 @LineBreak
  A = (lb*r-l*(r-b))/2 @LineBreak
  @LineBreak
  @Bold Périmètre : U = lb+l @LineBreak

@Section Consommation mémoire d'une image

  a = Largeur de l'image (en Pixel) @LineBreak
  b = Hauteur de l'image (en Pixel) @LineBreak
  bits = Profondeur de couleurs de l'image (en Bits) @LineBreak
  @LineBreak
  taille = a * b * (bits / 8) @LineBreak
  
  
@Section Distance entre deux points

  (x1|y1) = coordonnées XY du point n°1 @LineBreak
  (x2|y2) = coordonnées XY du point n°2 @LineBreak
  a = Distance entre les points @LineBreak
  @LineBreak
  a = SQR((x1-x2)^2 + (y1-y2)^2) @LineBreak


@Section "Mélange" de deux valeurs
  
  a = 1er Nombre @LineBreak
  b = 2ème Nombre @LineBreak
  v = Rapport (0% - 100%) @LineBreak
  e = Résultat @LineBreak
  @LineBreak
  e = ((a*v)/100) + ((b*(100-v)) / 100) @LineBreak
