@Title Migration de PureBasic 5.50 vers 5.60

@Section Cipher library

@LineBreak
Base64Encoder: Fonction renommée
@Code
  ; Ancien
  Base64Encoder()

  ; Nouveau
  Base64EncoderBuffer()
@EndCode

Base64Decoder: Fonction renommée
@Code
  ; Ancien
  Base64Decoder()

  ; Nouveau
  Base64DecoderBuffer()
@EndCode

@Section Autres
  La syntaxe autonome 'Define.b', pour changer le type par défaut des variables non typées, 
  est désormais interdite et lèvera une 'Erreur de syntaxe'. Il suffit de supprimer 
  cette instruction de vos code tout simplement (assurez-vous de typer vos variables 
  non typées convenablement).@LineBreak 
  Remarque: La syntaxe 'Define.b a, b, c' est toujours pris en charge.
