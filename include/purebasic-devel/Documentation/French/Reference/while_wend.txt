@Title While : Wend

@Syntax

  @Keyword While <expression>
    ...
  @Keyword Wend

@Description

  @Keyword While produit une boucle jusqu'à ce que l'expression devienne fausse. 
  Il est à noter qu'avec le test accompagnant @Keyword While, si la première tentative échoue, 
  le programme n'entrera pas dans la boucle et évitera donc cette section de code contrairement 
  à une boucle Repeat qui est toujours exécutée au moins une fois (car le test est effectué en sortie 
  de code conditionnel).

@Example

@Code
  b = 0
  a = 10
  While a = 10 
    b = b+1 
    If b=10 
      a=11 
    Endif 
  Wend 
@EndCode  

  Ce programme boucle jusqu'à ce que la valeur "a" devienne différente de 10. La boucle sera donc exécutée 10 fois.
