@Library Desktop

@Overview
  Die Desktop (= Arbeitsoberfläche Ihres Betriebssystems) Bibliothek ermöglicht den Zugriff auf Informationen
  über die Desktopumgebung des Anwenders, wie Bildschirmbreite und -höhe, Farbtiefe, Mausposition, etc. 

@CommandList

@ExampleFile All Desktop.pb

@SupportedOS

;--------------------------------------------------------------------------------------------------------

@Function Ergebnis = ExamineDesktops()

@Description
  Ermittelt Informationen über alle am lokalen Computer angeschlossenen Desktops.
  Dieser Befehl muss vor der Benutzung folgender Befehle aufgerufen werden: @@DesktopDepth, 
  @@DesktopFrequency, @@DesktopHeight, 
  @@DesktopName und @@DesktopWidth.

@NoParameters

@ReturnValue
  Gibt im Erfolgsfall die Anzahl der vorhandenen Desktops zurück,
  andernfalls Null.

@Remarks
  Typischerweise hat der "Normalanwender" nur einen Desktop,
  aber Multi-Bildschirm-Anwender können mehrere Desktops haben, jeder mit seiner eigenen Auflösung.

@Example
@Code
  MessageRequester("Desktop Information", "Sie haben "+Str(ExamineDesktops())+" Desktops.")
@EndCode

@SeeAlso
  @@DesktopDepth, @@DesktopFrequency,
  @@DesktopHeight, @@DesktopName, @@DesktopWidth

@SupportedOS

;--------------------------------------------------------------------------------------------------------

@Function Ergebnis = DesktopDepth(#Desktop)

@Description
  Gibt die Farbtiefe des angegebenen Desktops zurück.

@Parameter "#Desktop"
  Der Index des Desktops. Der erste Index spezifiziert immer den Hauptdesktop (-monitor).
  Der erste Index-Wert ist Null.

@ReturnValue
  Gibt die Farbtiefe in Bits pro Pixel zurück: 1, 2, 4, 8, 15, 16, 24 oder 32 Bit.

@Remarks
  @@ExamineDesktops muss vor diesem Befehl aufgerufen werden, um die Informationen über
  vorhandene Desktops zu ermitteln.

@Example
@Code
  ExamineDesktops()
  MessageRequester("Display Information", "Aktuelle Auflösung = "+Str(DesktopWidth(0))+"x"+Str(DesktopHeight(0))+"x"+Str(DesktopDepth(0)))
@EndCode

@SeeAlso
  @@ExamineDesktops,
  @@DesktopFrequency,
  @@DesktopHeight, @@DesktopName, @@DesktopWidth

@SupportedOS

;--------------------------------------------------------------------------------------------------------

@Function Ergebnis.d = DesktopResolutionX()

@Description
  Gibt den DPI-Auflösungsfaktor des Desktops auf der 'x'-Achse zurück.

@NoParameters

@ReturnValue
  Gibt den DPI-Auflösungsfaktor des Desktops auf der 'x'-Achse zurück. Wenn der Wert '1' lautet, wurde
  auf die Anzeige auf der 'x'-Achse kein DPI-Faktor angewendet.
  Wenn der Wert '1.25' lautet, wurde der Anzeige auf der 'x'-Achse ein Faktor von 125% zugewiesen.

@Remarks
  Die Anwendung muss mit der @ReferenceLink "ide_compiler" "Option 'DPI-kompatibel'" kompiliert werden,
  damit dieser Befehl einen tatsächlichen DPI-Auflösungsfaktor liefert.
  Wenn nicht, ist das Ergebnis immer '1'.

@Example
@Code
  Debug "Desktop DPI 'x' Faktor: " + DesktopResolutionX()
@EndCode

@SeeAlso
  @@DesktopResolutionY, @@DesktopScaledX, @@DesktopScaledY, @@DesktopUnscaledX, @@DesktopUnscaledY

@SupportedOS

;--------------------------------------------------------------------------------------------------------

@Function Ergebnis = DesktopScaledX(Wert)

@Description
  Gibt den skalierten Wert gemäß der aktuellen DPI-Anzeige auf der x-Achse zurück. Dies ist vor allem
  nützlich, um die reale Pixelposition unabhängig von der angezeigten DPI zu berechnen.

@Parameter "Value"
  Der zu verwendende Wert.

@ReturnValue
  Gibt den skalierten Wert gemäß der aktuellen DPI-Anzeige auf der x-Achse zurück. Beispielsweise
  resultiert bei einer Anzeige mit einem zugewiesenen DPI von 125% aus einem Wert von 100 im Ergebnis 125.

@Remarks
  Die Anwendung muss mit der @ReferenceLink "ide_compiler" "Option 'DPI-kompatibel'" kompiliert werden,
  damit dieser Befehl tatsächlich einen skalierten DPI-Wert zurückgibt.
  Wenn nicht, ist das Ergebnis immer gleich dem 'Wert' Parameter.

@Example
@Code
  Debug "Desktop DPI 'x' skalierter Wert von 100: " + DesktopScaledX(100)
@EndCode

@SeeAlso
  @@DesktopResolutionX, @@DesktopResolutionY, @@DesktopScaledY, @@DesktopUnscaledX, @@DesktopUnscaledY

@SupportedOS

@Function Ergebnis = DesktopFrequency(#Desktop)

@Description
  Gibt die Frequenz des angegebenen Desktops zurück.

@Parameter "#Desktop"
  Der Index des Desktops. Der erste Index spezifiziert immer den Hauptdesktop (-monitor).
  Der erste Index-Wert ist Null.

@ReturnValue
  Gibt die Frequenz des angegebenen Desktops in Hertz zurück.
  Ist das Ergebnis gleich 0, dann wird die Standard Hardware-Frequenz verwendet (der
  Anwender hat diese nicht gegenüber der Standardeinstellung verändert), oder die
  aktuelle Frequenz konnte nicht ermittelt werden.

@Remarks
  @@ExamineDesktops muss vor diesem Befehl aufgerufen werden, um die Informationen über
  vorhandene Desktops zu ermitteln.
  @Bold "Hinweis:" Auf Linux gibt diese Funktion immer 0 zurück.

@Example
@Code
  ExamineDesktops()
  f = DesktopFrequency(0)
  If f = 0
    MessageRequester("Display Information", "Es wurde keine Desktop-Frequenz eingestellt, die Standard-Hardwarefrequenz wird verwendet.")
  Else
    MessageRequester("Display Information", "Frequenz des Desktops: "+Str(f)+" Hz.")
  EndIf
@EndCode

@SeeAlso
  @@ExamineDesktops,
  @@DesktopDepth,
  @@DesktopHeight, @@DesktopName, @@DesktopWidth

@SupportedOS

@Function Ergebnis = DesktopX(#Desktop)

@Description
  Gibt die x-Koordinate des angegebenen Desktops zurück.

@Parameter "#Desktop"
  Der Index des Desktops. Der erste Index spezifiziert immer den Hauptdesktop (-monitor).
  Der erste Index-Wert ist Null.

@ReturnValue
  Gibt die x-Koordinate (in Pixel) der linken oberen Ecke des angegebenen Desktops zurück.
  Die Koordinate ist relativ zur linken oberen Ecke des Hauptmonitors. Sie ist negativ,
  wenn sich der angebenene Monitor links vom Hauptmonitor befindet.

@Remarks
  @@ExamineDesktops muss vor diesem Befehl aufgerufen werden, um die Informationen über
  vorhandene Desktops zu ermitteln.

@SeeAlso
  @@ExamineDesktops,
  @@DesktopDepth, @@DesktopY, 
  @@DesktopHeight, @@DesktopWidth

@SupportedOS

@Function Ergebnis = DesktopMouseX()

@Description
  Gibt die absolute x-Position der Maus auf dem Desktop zurück.

@NoParameters

@ReturnValue
  Gibt die x-Koordinate (in Pixel) der Maus relativ zur linken oberen Ecke des Hauptmonitors zurück.
  Die Koordinate ist negativ, wenn sich die Maus auf einem Monitor links vom Hauptmonitor befindet.

@Example
@Code
  If OpenWindow(0, 0, 0, 300, 30, "Desktop Maus-Monitor", #PB_Window_SystemMenu | #PB_Window_ScreenCentered)
    TextGadget(0, 10, 6, 200, 20, "")
    
    Repeat
      Event = WindowEvent()
      
      If Event = 0 ; Es sind keine Ereignisse mehr in der Warteschlange, daher halten wir den Prozess
                   ; für ein paar Millisekunden an, um das Multitasking nicht zu beeinträchtigen
        SetGadgetText(0, "Desktop Maus-Position: "+Str(DesktopMouseX())+","+Str(DesktopMouseY()))
        Delay(20)
      EndIf
       
    Until Event = #PB_Event_CloseWindow
  EndIf
@EndCode

@SeeAlso
  @@DesktopMouseY, @@DesktopX, @@DesktopWidth, @@WindowMouseX

@SupportedOS

;--------------------------------------------------------------------------------------------------------
