@Library Runtime

@Overview

  Auf Runtime-Objekte (in Deutsch auch "Laufzeit-Objekte") kann durch Verwendung ihrer String-Referenzen
  auch dann zugegriffen werden, wenn das Programm kompiliert wurde, 
  Für weitere Informationen über das "Runtime"-Konzept siehe @ReferenceLink "Runtime" "Runtime".
  
@CommandList

@ExampleFile All Runtime.pb

@SupportedOS

;--------------------------------------------------------------------------------------------------------

@Function Ergebnis = GetRuntimeInteger(Objekt$)

@Description
  Gibt den Integer-Wert des Runtime-Objekts zurück. Ist das Runtime-Objekt eine Prozedur,
  wird die Prozedur-Adresse zurückgegeben.

@Parameter "Objekt$"
  Name des Objekts, von dem der Wert ermittelt werden soll. Die folgenden Objekte werden
  unterstüzt: @LineBreak
- Variable: der Objektname ist 'VariablenName' (ohne Beachtung von Groß- und Kleinschreibung). @LineBreak
- Konstante: der Objektname ist '#KonstantenName' (ohne Beachtung von Groß- und Kleinschreibung). @LineBreak
- Prozedur: der Objektname ist 'ProzedurName()' (ohne Beachtung von Groß- und Kleinschreibung). 

@ReturnValue
  Gibt den Integer-Wert des angegebenen Objekts zurück - oder Null, wenn es nicht gefunden wurde.
  Da Null ein gültiger Integer-Wert ist, kann @@IsRuntime verwendet werden, um zu ermitteln, 
  ob das Runtime-Objekt wirklich existiert. Ist die Variable vom Typ Float oder Double, wird
  sie automatisch in Integer umgewandelt. Ist das Runtime-Objekt eine Prozedur,
  wird die Prozedur-Adresse zurückgegeben.

@SeeAlso
  @@SetRuntimeInteger, @@IsRuntime

@SupportedOS

;--------------------------------------------------------------------------------------------------------

@Function Ergebnis$ = GetRuntimeString(Objekt$)

@Description
  Gibt den String-Wert des Runtime-Objekts zurück.

@Parameter "Objekt$"
  Name des Objekts, von dem der Wert ermittelt werden soll. Die folgenden Objekte
  werden unterstützt: @LineBreak
- Variable: der Objektname ist 'VariablenName' (ohne Beachtung von Groß- und Kleinschreibung). @LineBreak

@SeeAlso
  @@SetRuntimeString, @@IsRuntime

@SupportedOS

@Function Ergebnis = IsRuntime(Objekt$)

@Description
  Überprüft ob das Objekt als "Runtime"-Objekt deklariert wurde.

@Parameter "Objekt$"
  Name des zu überprüfenden Objekts. Die folgenden Objekte werden unterstützt: @LineBreak
- Variable: der Objektname ist 'VariablenName' (ohne Beachtung von Groß- und Kleinschreibung). @LineBreak
- Konstante: der Objektname ist '#KonstantenName' (ohne Beachtung von Groß- und Kleinschreibung). @LineBreak
- Prozedur: der Objektname ist 'ProzedurName()' (ohne Beachtung von Groß- und Kleinschreibung). 
  @LineBreak
  Beim Zugriff auf öffentliche @ReferenceLink "module" "Modul"-Elemente ist das Modul-Präfix obligatorisch,
  auch wenn @Keyword UseModule verwendet wird.

@ReturnValue
  Gibt einen Wert ungleich Null zurück, wenn das Objekt als @ReferenceLink "Runtime" "Runtime" deklariert
  wurde, und andernfalls Null.

@SupportedOS
