Einführung und Voraussetzungen

Der Sinn dieses Kapitels liegt darin, Ihnen einen fließenden Start in das Schreiben von Software mit PureBasic zu geben.

Die Grundlagen der Sprache werden beschrieben und dann wird ein Beispiel aufgeführt, das Sie eingeben und ausführen können. Dies ermöglicht Ihnen, sich an die PureBasic Entwicklungsumgebung zu gewöhnen und sicherzustellen, dass es korrekt installiert ist und richtig läuft. Anweisungen, die häufig überall im Benutzerhandbuch verwendet werden, werden im Detail an diesem ersten Beispiel demonstriert und erklärt.

Zuletzt wird das Beispiel Zeile für Zeile auseinandergenommen, damit Sie verstehen, was passiert (auf einem grundlegenden Niveau).

Die einzigen Voraussetzungen für dieses Kapitel sind, dass Sie PureBasic installiert haben.
