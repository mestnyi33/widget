Weitere Informationen: Hinweise

Wir sind jetzt am Ende des Benutzerhandbuchs angekommen. Hoffentlich haben Sie jetzt ein Gefühl für die Sprache erlangt und sind bereit, einige eigene Projekte zu starten. Hier finden Sie einige Dinge, an die Sie sich erinnern sollten, bevor Sie starten:

  • Es hilft manchmal, wenn Sie wissen, welches Programm Sie schreiben wollen, bevor Sie mit dem Schreiben beginnen. Sie haben dann eine Vorstellung davon, welche Dinge das Programm enthalten soll und haben ein Ziel.

  • Planen Sie die Software, die Sie erschaffen wollen. Schreiben Sie sich Notizen, Ideen und die Schritte, die Ihr Programm gehen soll, entweder elektronisch oder auf ein Stück Papier auf und Sie werden es leichter finden, sich Ihren Code vorzustellen.

  • Werfen Sie einen Blick auf die nächste Seite, die eine kurze Beschreibung darüber gibt, was die PureLibraries alles können. Sie sollte Ihnen einen Überblick über die Power des vollständigen PureBasic Pakets geben.

  • Die beste Art zu lernen ist es zu tun, fangen Sie also an! Werfen Sie einen Blick auf die mit PureBasic mitgelieferten Beispiele, wann immer Sie können, selbst wenn sie sich gerade nicht darauf beziehen, woran Sie arbeiten. Sie könnten Ihnen einige allgemeine Ideen darüber geben, wie man etwas lösen kann.

  • Erinnern Sie sich daran - PureBasic ist sowohl für Anfänger wie auch für Power-Anwender geeignet. Was Sie damit tun, liegt letztlich bei Ihnen - Sie entscheiden, wie fortgeschritten Sie Dinge erledigen wollen.
