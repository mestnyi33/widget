Grundlagen: Wie man PureBasic erhalten kann

Wie erhalten Sie PureBasic?

Es gibt drei Möglichkeiten, wie Sie PureBasic bekommen können. Die erste ist eine eingeschränkte Demonstrationsversion, die nur eine gewisse Anzahl an Zeilen des Codes kompilieren kann, keinen Zugriff auf sämtliche Befehlsbibliotheken hat und immer den Debugger aktiviert hat (deshalb werden Ihre Programme nicht mit voller Geschwindigkeit laufen). Diese ist frei verfügbar auf der PureBasic Webseite - http://www.purebasic.com.

Die zweite Möglichkeit, PureBasic zu erhalten, besteht darin, es entweder per Scheck (oder mittels Überweisung an das deutsche Team-Mitglied) oder über das sichere Online-Einkaufssystem auf der PureBasic Webseite zu kaufen - http://www.purebasic.com. Der Kauf ermöglicht Ihnen den vollen Zugang zur gesamten Power von PureBasic, berechtigt Sie zum Erhalt der kostenlosen Updates und für einen Preis erhalten Sie die PureBasic Vollversionen für mehrere Betriebssysteme (Windows, Linux und Mac OS X). Mit dieser Methode helfen Sie auch, die weitere Entwicklung von PureBasic zu sichern. Wenn Sie PureBasic "offline" (z.B. per Scheck) bestellen und über keinen Internet-Zugang verfügen, dann wird Ihnen (allerdings mit einer Lieferzeit von 2-3 Wochen) eine CD mit dem PureBasic-Paket zugesandt. Diese enthält genau das Gleiche wie die "Online" Version von PureBasic, wofür Ihnen eine Internet-Adresse zum Herunterladen zugesandt wird.

Die letzte Methode zum Erhalt von PureBasic besteht darin, die Vollversion zu einem reduzierten Preis im Einzelhandel zu kaufen. Diese bietet alle Leistungen der zweiten Methode sowie eines reduzierten Preises, hat aber den Nachteil, dass Sie nicht zum Erhalt der kostenlosen Updates berechtigt sind. Ein Upgrade wird jedoch auf der Bestellseite von http://www.purebasic.com angeboten.
