Einführung

Willkommen zum PureBasic Benutzerhandbuch - dem Abschnitt des Handbuchs, welcher sowohl unerfahrene Programmierer als auch Neulinge in PureBasic durch die Sprache führen soll. Obwohl dieses Benutzerhandbuch für Programmierneulinge lesbar sein soll, hoffen wir, dass auch erfahrene Programmierer sich die Zeit zum Durchschauen nehmen - Sie sollten dadurch schnelle Fortschritte machen können.

Das Benutzerhandbuch beginnt mit den einfachsten Bestandteilen der PureBasic-Sprache und bahnt sich seinen Weg durch die verschiedenen Themen und Befehle, in dem es mit Hilfe von Beispielen und Erklärungen jener Beispiele Ihnen volles Verständnis für das Schreiben von Software in PureBasic vermittelt. Das Benutzerhandbuch erklärt, warum, wann und wo Sie verschiedene Teile von Programmcode verwenden müssen und was Sie tun, wenn Sie diesen Code verwenden.

Dieses erste Kapitel ist eine Einführung sowohl in PureBasic (in Bezug auf ein "Stück" Software, noch nicht speziell auf das Schreiben von Software ausgerichtet) als auch den Rest dieses Benutzerhandbuchs. In den folgenden Themenabschnitten finden Sie Erklärungen, wie Sie PureBasic kaufen, installieren, starten und verwenden können. Der Inhalt des restlichen Benutzerhandbuchs wird auch beschrieben, so dass Sie wissen, was zu Sie erwarten können.

Am unteren Ende jeder Seite des Benutzerhandbuchs finden Sie drei Links. Der erste bringt Sie zurück zum vorherigen Thema in einem Kapitel. Der dritte verzweigt zum nächsten Thema in einem Kapitel. Der zweite Link kehrt zur Inhaltsseite zurück. Um zum nächsten Thema in dieser Einführung zu gelangen, klicken Sie unten auf den Link "Über PureBasic".
