Einfache Verarbeitung: Praktische Übungen

Diese Übungen werden mitgeliefert, damit Sie Ihr Verständnis - was die mathematischen Operatoren tun und wann sie verwendet werden sollten - testen können. Sie testen auch Ihr Wissen der vorherigen Kapitel (Grundlagen, allgemeine Befehle und Variablen). Beispiellösungen werden angeboten, sind aber wirklich nur ein Beispiel für eine Möglichkeit zur Lösung des Problems. Es gibt wahrscheinlich viele Wege, jede Übung zu beenden. Machen Sie sich also keine Gedanken, wenn Ihre Lösung nicht genau übereinstimmt.

Jede Lösung erklärt in den Kommentaren, was passiert und warum etwas gemacht wird.

4.1) Schreiben Sie ein Programm, um die Fläche eines Rechtecks zu berechnen und anzuzeigen. Sie sollten Variablen verwenden, um die Breite, die Höhe und die Fläche zu speichern. Aber Sie sollten die Werte der Breite- und Höhevariablen direkt in Ihrem Code einsetzen, anstelle eine anderen Weg (wie vom Anwender eingeben zu lassen) zu wählen. (Lösung)

4.2) Erweitern Sie das von Ihnen für die Übung 4.1 geschriebene Programm, um die Fläche der Wände in einem Zimmer zu berechnen. Verwenden Sie eine weitere Variable Länge zusammen mit den vorherigen. (Lösung)

4.3) Modifizieren Sie das von Ihnen für die Übung 4.2 geschriebene Programm, um die Anzahl der Eimer von Farbe zu berechnen, die benötigt werden, um ein Zimmer zu streichen. (Hinweis: Sie sollten eine weitere Variable haben, die die Fläche speichert, welche mit einem Eimer an Farbe abgedeckt werden kann). (Lösung)

4.4) Schreiben Sie ein Programm, um den Durchschnitt der Prüfungsergebnisse (ausgehend von 100) von fünf Studenten zu berechnen. Verwenden Sie eine Variable für jedes Ergebnis und eine für den Durchschnitt. Geben Sie jedes Ergebnis und den Durchschnitt aus. (Lösung)

4.5) Schreiben Sie ein Programm, um eine Aufstellung der Details für ein Mitglied eines Klubs auszudrucken. Sie sollten separate Variablen zum Speichern von Vorname und Nachname der Person, Hausnummer, Straßenname, Stadt und Mitgliedsnummer verwenden. Die Hausnummer und Mitgliedsnummer sollten in "Long" Variablen gespeichert werden. Das Format der Anzeige sollte wie folgt aussehen: 

<Vorname> <Nachname> : <Mitgliedsnummer>
<Hausnummer> <Straßenname>
<Stadt>
(Lösung)
