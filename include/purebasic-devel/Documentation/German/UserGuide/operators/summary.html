Einfache Verarbeitung: Zusammenfassung

Dieses Kapitel hat Sie in die Handhabung von Operatoren eingeführt, welches ein oder zwei spezielle Zeichen in Ihrem Quellcode sind, um eine einfache Verarbeitung auszuführen. Die mathematischen Operatoren wurden in diesem Kapitel beschrieben, weitere Operatoren folgen in zukünftigen Kapiteln. Die Hauptpunkte, die Sie aus diesem Kapitel mitnehmen sollten, sind:

  • Operatoren sind eine einfache Möglichkeit zur Verarbeitung von Werten und werden normalerweise durch ein oder zwei spezielle Zeichen in Ihrem Quellcode dargestellt.

  • Die Zuweisungs- (Gleich, =) und Additions- (+) Operatoren sind die einzigen, welche zusammen mit Strings benutzt werden können. Alle Operatoren arbeiten mit numerischen Werten.

  • Die meisten Operatoren arbeiten auch in einer Kurzform, wie folgt <Variable> <Operator> <Ausdruck>. Dies ist dasselbe, als wenn Sie <Variable> = <Variable> <Operator> <Ausdruck> in Ihrem Code schreiben.

  • Wenn Sie den Abkürzungsmodus verwenden, darf das Ergebnis der Berechnung nicht verwendet werden, andernfalls wird die Variable nicht modifiziert.

  • Ausdruck ist der allgemeine Name für eine Berechnung und kann so einfach sein, wie ein fester Wert oder so komplex, wie eine Formel, die viele Operatoren und Elemente - welche einen Wert (einschließlich z.B. dem von einem Befehl zurückgegebenen Wert, wenn Sie den Befehl direkt im Ausdruck verwenden) repräsentieren - benutzt.

  • Klammern können verwendet werden, um die Reihenfolge der Auswertung von Ausdrücken zu ändern, welche normalerweise der Reihenfolge Multiplikation oder Division gefolgt von Addition oder Subtraktion folgt. Operatoren desselben Ranges werden in der Reihenfolge beurteilt, wie sie in Ihrem Quellcode geschrieben wurden (von links nach rechts).
