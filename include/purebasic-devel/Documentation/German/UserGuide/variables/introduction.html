Variablen: Einführung

Das vorherige Kapitel beschrieb die Grundlagen der PureBasic Sprache. Dieses Kapitel beschreibt jetzt einen der Grundbausteine von jedem Stück Software: die Variablen. Es führt Sie von warum und wann Variablen gebraucht werden, über die Typen von Variablen, die Sie in PureBasic verwenden können, bis wie man sie erstellen kann.

Voraussetzung für dieses Kapitel ist, dass Sie PureBasic erfolgreich installiert und laufen haben. Sie sollten die Grundlagen der Sprache und die typischen Befehle für das Anzeigen von Text auf der Konsole vom vorherigen Kapitel kennen.
