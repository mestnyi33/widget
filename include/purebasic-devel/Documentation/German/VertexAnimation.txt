@Library VertexAnimation

@Overview
  Vertex-Animation ermöglicht die Steuerung der Animations-Posen (auch Haltung, Stellung)
  von Meshes. Das @LibraryLink "mesh" "Mesh" muss eine Liste vordefinierter Posen enthalten.

@CommandList

@ExampleFile All VertexAnimation.pb

@SupportedOS

;--------------------------------------------------------------------------------------------------------

@Function Ergebnis = CreateVertexAnimation(#Mesh, Animation$, Länge)

@Description
  Erstellt eine neue Vertex-Animation für das angegebene Mesh.

@Parameter "#Mesh"
  Das zu verwendende Mesh, um die neue Animation zu erstellen.

@Parameter "Animation$"
  Der neue Animations-Name. Der Name beachtet die Groß- und Kleinschreibung.
  
@Parameter "Länge"
  Die Länge der neuen Animation (in Millisekunden).

@ReturnValue
  Gibt einen Wert ungleich Null zurück, wenn die Animation erfolgreich erstellt wurde.

@SeeAlso
  @@CreateVertexTrack

@SupportedOS

;--------------------------------------------------------------------------------------------------------

@Function Ergebnis = CreateVertexTrack(#Mesh, Animation$, Index)

@Description
  Erstellt einen neuen Track (Spur, Strecke) mit der Vertex-Animation für das angegebene Mesh.
  Die Animation sollte bereits mit @@CreateVertexAnimation erstellt worden sein, oder vordefiniert
  im Mesh. Jeder Track hat die gleiche Länge, wie bei @@CreateVertexAnimation definiert.
  
@Parameter "#Mesh"
  Das zu verwendende Mesh, um den neuen Animations-Track zu erstellen.

@Parameter "Animation$"
  Der Animations-Name. Dieser Name beachtet die Groß- und Kleinschreibung.
  Die Animation sollte bereits mittels @@CreateVertexAnimation erstellt worden sein.
  
@Parameter "Index"
  Der neue Index des Tracks. Der Index-Wert sollte bei Null starten.

@ReturnValue
  Gibt einen Wert ungleich Null zurück, wenn der Animations-Track erfolgreich erstellt wurde.

@SeeAlso
  @@CreateVertexAnimation

@SupportedOS

;--------------------------------------------------------------------------------------------------------

@Function Ergebnis = CreateVertexPoseKeyFrame(#Mesh, Animation$, Track, Zeit)

@Description
  Erstellt ein neues Keyframe (auch "Schlüsselbild") in der Vertex-Animation.
  Die Animation sollte bereits mit @@CreateVertexAnimation erstellt worden sein, oder vordefiniert
  im Mesh.
  
@Parameter "#Mesh"
  Das zu verwendende Mesh.

@Parameter "Animation$"
  Der Animations-Name. Dieser Name beachtet die Groß- und Kleinschreibung.
  Die Animation sollte bereits mittels @@CreateVertexAnimation erstellt worden sein.
  
@Parameter "Track"
  Der Track-Index. Der Track (oder auch Spur, Strecke) muss zuvor mit @@CreateVertexTrack
  erstellt worden sein.

@Parameter "Zeit"
  Der Zeitpunkt (in Millisekunden) innerhalb der Animation, zu dem das Keyframe festgelegt
  werden soll. Dieser Wert muss zwischen Null und der bei @@CreateVertexAnimation definierten
  'Länge' liegen.

@ReturnValue
  Gibt den neuen Keyframe-Index zurück.

@SeeAlso
  @@CreateVertexAnimation, @@CreateVertexTrack

@SupportedOS

;--------------------------------------------------------------------------------------------------------

@Function AddVertexPoseReference(#Mesh, Animation$, Track, Keyframe, PoseIndex, Einfluss)

@Description
  Fügt einen neuen Posen-Bezug (auch Referenz auf eine Pose) zur Animation hinzu.
  Die Animation sollte bereits mittels @@CreateVertexAnimation erstellt worden sein.
  
@Parameter "#Mesh"
  Das zu verwendende Mesh.

@Parameter "Animation$"
  Der Animations-Name. Dieser Name beachtet die Groß- und Kleinschreibung.
  Die Animation sollte bereits mittels @@CreateVertexAnimation erstellt worden sein.
  
@Parameter "Track"
  Der Track-Index. Der Track (oder auch Spur, Strecke) muss zuvor mit @@CreateVertexTrack
  erstellt worden sein.

@Parameter "Keyframe"
  Der Keyframe-Index. Das Keyframe muss zuvor mittels @@CreateVertexPoseKeyFrame
  erstellt worden sein.

@Parameter "PoseIndex"
  Der Posen-Index. Der erste Index-Wert für eine Pose beginnt bei Null.

@Parameter "Einfluss"
  Der Einfluss (oder auch das Gewicht) der Pose. Dieser Wert reicht von 0.0 (keinen
  Einfluss) bis 1.0 (voller Einfluss).

@NoReturnValue

@SeeAlso
  @@CreateVertexAnimation, @@CreateVertexTrack

@SupportedOS

;--------------------------------------------------------------------------------------------------------

@Function UpdateVertexPoseReference(#Mesh, Animation$, Track, Keyframe, PoseIndex, Einfluss)

@Description
  Aktualisiert einen neuen Posen-Bezug (auch Referenz auf eine Pose) der Animation.
  Die Animation sollte bereits mittels @@CreateVertexAnimation erstellt worden oder
  vordefiniert im Mesh sein.
  
@Parameter "#Mesh"
  Das zu verwendende Mesh.

@Parameter "Animation$"
  Der Animations-Name. Dieser Name beachtet die Groß- und Kleinschreibung.
  Die Animation sollte bereits mittels @@CreateVertexAnimation erstellt worden sein.
  
@Parameter "Track"
  Der Track-Index. Der Track (oder auch Spur, Strecke) muss zuvor mit @@CreateVertexTrack
  erstellt worden sein.

@Parameter "Keyframe"
  Der Keyframe-Index. Das Keyframe muss zuvor mittels @@CreateVertexPoseKeyFrame
  erstellt worden sein.

@Parameter "PoseIndex"
  Der Posen-Index. Der erste Index-Wert für eine Pose beginnt bei Null.

@Parameter "Influence"
  Der neue Einfluss (oder auch das Gewicht) der Pose. Dieser Wert reicht von 0.0 (keinen
  Einfluss) bis 1.0 (voller Einfluss).

@NoReturnValue

@SeeAlso
  @@CreateVertexAnimation, @@CreateVertexTrack

@SupportedOS

;--------------------------------------------------------------------------------------------------------

@Function Ergebnis = VertexPoseReferenceCount(#Mesh, Animation$, Track, Keyframe)

@Description
  Gibt die Anzahl von Posen-Bezügen (oder auch Referenzen auf Posen) im angegebenen Keyframe zurück.
  Die Animation sollte bereits mittels @@CreateVertexAnimation erstellt worden oder
  im Mesh vordefiniert sein.
  
@Parameter "#Mesh"
  Das zu verwendende Mesh.

@Parameter "Animation$"
  Der Animations-Name. Dieser Name beachtet die Groß- und Kleinschreibung.
  Die Animation sollte bereits mittels @@CreateVertexAnimation erstellt worden sein.
  
@Parameter "Track"
  Der Track-Index. Der Track (oder auch Spur, Strecke) muss zuvor mit @@CreateVertexTrack
  erstellt worden sein.

@Parameter "Keyframe"
  Der Keyframe-Index. Das Keyframe muss zuvor mittels @@CreateVertexPoseKeyFrame
  erstellt worden sein.

@ReturnValue
  Gibt die Anzahl von Posen-Bezügen im angegebenen Keyframe zurück.

@SeeAlso
  @@CreateVertexAnimation, @@CreateVertexTrack, @@AddVertexPoseReference

@SupportedOS

;--------------------------------------------------------------------------------------------------------

@Function Ergebnis = MeshPoseCount(#Mesh)

@Description
  Gibt die Anzahl von Posen im Mesh zurück. Eine Pose ist eine vordefinierte
  Vertex-Animation im Mesh.
  
@Parameter "#Mesh"
  Das zu verwendende Mesh.

@ReturnValue
  Gibt die Anzahl von Posen im angegebenen Mesh zurück.

@SeeAlso
  @@MeshPoseName, @@AddVertexPoseReference

@SupportedOS

;--------------------------------------------------------------------------------------------------------

@Function Ergebnis$ = MeshPoseName(#Mesh, PoseIndex)

@Description
  Gibt den Namen der Pose im Mesh zurück. Eine Pose ist eine vordefinierte
  Vertex-Animation im Mesh.
  
@Parameter "#Mesh"
  Das zu verwendende Mesh.

@Parameter "PoseIndex"
  Der Posen-Index, von dem der Name ermittelt werden soll. Der erste Index-Wert für
  eine Pose beginnt bei Null. Der Index muss niedriger als das Ergebnis von
  @@MeshPoseCount sein.

@ReturnValue
  Gibt den Namen der Pose beim angegebenen Index zurück.

@SeeAlso
  @@MeshPoseName, @@AddVertexPoseReference

@SupportedOS
